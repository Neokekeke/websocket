# websocket
websocket实时聊天推送


参考文档：socket.io

https://socket.io/get-started/chat/

#### 1、项目运行方式：

npm install

#### 2、node 运行app.js

node app.js

#### 3、项目监听的端口是：8082，在浏览器中打开：

http://localhost:8082/

### 项目使用了node的express，socket.io模块，参考配置如下

#### 项目配置文件初始化：

npm init 

#### express 安装： 

npm install express --save

#### socket.io 安装：

npm install socket.io --save
